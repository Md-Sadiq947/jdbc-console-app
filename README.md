# JDBC Person Management Console App

A console-based Java application using JDBC to manage person records in PostgreSQL.

## Features
- Add new person
- View all persons
- Update person details
- Delete person
- Menu-driven interface

## Technologies
- Java
- JDBC
- PostgreSQL
- Maven

## How to Run
1. Clone the repo: `git clone https://github.com/Md-Sadiq947/jdbc-console-app.git`
2. Setup PostgreSQL (create database 'test' and 'person' table)
3. Update username/password in code
4. Run Main.java
